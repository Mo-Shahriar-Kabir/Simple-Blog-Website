# Simple Blog Website

A minimal static blog built with plain HTML, CSS and a tiny JS file.

Files
- index.html — main page
- style.css — styles
- index.js — small interactivity
- images/ — media

Preview
- Open index.html in your browser
- Or serve with: python -m http.server 8000 and visit http://localhost:8000

Edit
- Change posts in index.html
- Update styles in style.css
- Add scripts to index.js

Contribute
- Fork, edit, and open a PR.
